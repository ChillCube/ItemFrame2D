# ItemFrame2D

A node to visualize items on the screen, useful for inventory UI

## Installation

Make sure to download the ChillCube Developer tools to use this addon: https://github.com/ChillCube/ChillCube-Developer-Tools.git

### Linux
To add this to your project, copy paste these commands into terminal at the root of your project:
```bash
clone-gd-addon https://github.com/ChillCube/ItemFrame2D.git
```

## Documentation
You can find the documentation at: https://github.com/ChillCube/ItemFrame2D/blob/main/DOCUMENTATION.md

## Deinstallation
### Linux
Run this in the root of your godot project:
```bash
remove-gd-addon ItemFrame2D
```

## For Developers
### Push Changes
Run this at the root of your directory to push changes to any addons you have:
```bash
push-all-addons
```
